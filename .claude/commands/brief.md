# Брифинг перед звонком с клиентом

Подготовь краткий брифинг для звонка с клиентом.

## Входные данные
$ARGUMENTS

(тип клиента: L1 / L2 / Pub / PubC, и любой дополнительный контекст)

## Что сделать

1. Прочитай `02-business-model/client-types.md` — понять профиль этого типа клиента
2. Прочитай `03-product/user-stories.md` — какие stories уже есть для этого типа
3. Прочитай `05-research/research-log.md` и релевантные анализы — что уже известно от похожих клиентов
4. Прочитай `03-product/product-strategy.md` — текущие приоритеты
5. Прочитай `04-bi/metrics-dictionary.md` — какие метрики доступны / планируются для этого типа

## Вывод

### Профиль типа клиента
- Что им важно
- Их типичные боли

### Что мы уже знаем (из предыдущих интервью)
- Подтверждённые боли
- Запрошенные фичи

### Гипотезы для проверки
- Что мы предполагаем, но ещё не валидировали для этого типа
- Какие user stories ещё без подтверждения

### Вопросы для звонка
5-7 конкретных вопросов, отсортированных по приоритету.
Фокус на том, чего мы ещё НЕ знаем.
