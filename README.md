# CAS Platform Documentation

База знаний для проектирования продукта CAS.AI — ad mediation платформы для мобильных разработчиков.

## Структура репозитория

| Папка | Содержимое |
|-------|-----------|
| `01-platform/` | Что такое CAS, компоненты, ограничения |
| `02-business-model/` | Бизнес-модель, типы клиентов, финмодель |
| `03-product/` | Продуктовые спецификации: кабинет, BI, онбординг, user stories |
| `04-bi/` | Справочник метрик и фильтров |
| `05-research/` | Исследования и интервью (структурированные анализы) |
| `assets/` | Мокапы, скриншоты конкурентов |

## Порядок чтения

1. `01-platform/platform-overview.md` — что такое CAS

## Конвенции

- Один документ = один уровень абстракции
- Ясность важнее полноты
- Markdown — единственный формат (без HTML/DOCX экспортов)
- Сырые транскрипты не хранятся — только структурированные анализы
