## Как пользоваться

1. Выбираем тему из бэклога → переводим статус в "В работе"
2. Производим hard content по процессу из `content-machine.md`
3. Нарезаем soft content → распространяем по owned media
4. После публикации → статус "Опубликовано", добавляем ссылки и метрики
