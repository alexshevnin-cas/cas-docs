# AI Context Rules

Purpose:
Define how AI agents should use this documentation.

General rules:
- Always start with README.md
- Respect abstraction levels
- Do not invent missing details

Context loading:
- Load only relevant documents for the task
- Do not assume future modules exist

If information is missing:
- Ask clarification questions
- Propose where a new document should be added

Writing style:
- Clear
- Structured
- No marketing language unless explicitly requested
